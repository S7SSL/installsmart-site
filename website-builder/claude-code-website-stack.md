## The Concept
Claude Code can build full, animated, professionally designed websites from a single prompt.
No templates. No drag-and-drop. No Squarespace. Just a prompt and a finished product.

## The 4-Tool Stack

1. **Claude Code** — writes entire website, file by file, from a single prompt
2. **Framer Motion** — real animations (page transitions, hover effects, scroll reveals)
3. **UI/UX Pro Max** — design skill giving Claude access to 50+ styles, 161 colour palettes, 57 font pairings (prevents generic AI output)
4. **21st.dev Magic** — library of 100+ polished React components (production-quality building blocks)

## Setup (5 mins, zero coding)

Requires Claude Code: https://docs.anthropic.com/en/docs/claude-code

```bash
claude install-skill https://github.com/tenfoldmarc/website-builder-setup
```

Then in Claude Code:
```
/website-builder-setup
```

Claude installs all three tools step by step, checks what's installed, and sets everything up.

## How It Works in Practice
- Open terminal
- Tell Claude: what the business does, who it's for, what vibe you want
- Claude builds layout, copy, animations, responsive design
- Live on screen in minutes
- Iterate instantly: "make the hero section bolder" → updates immediately
